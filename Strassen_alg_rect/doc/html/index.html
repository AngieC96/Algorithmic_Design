Description

Implementation of the Strassen's algorithm for matrix multiplication.

How to compile

Compilation

We include a standard Makefile for convenience, so you can run the usual make commands; out-of-source build are not supported.

It is also possible to generate the Makefile via cmake by opening a terminal in the source code folder and running 

cmake -G "Unix Makefiles" CMakeLists.txt
make

In both cases, if everything goes as it should, you now have an executable strassen_test.x in your build folder $BLD. You can run it via 

./strassen_test.x

Build the documentation

You can build the documentation by running 

make doc

The documentation is built into the folder $BLD/doc/, where $BLD is your build folder.
